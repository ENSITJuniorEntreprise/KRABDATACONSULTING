# KRABDATA CONSULTING - Site Web Vitrine

Site web vitrine professionnel pour KRABDATA CONSULTING, cabinet de conseil spécialisé dans la valorisation des données, la digitalisation et l'automatisation des processus métiers.

## 📋 Caractéristiques

- ✅ Site responsive (mobile, tablette, desktop)
- ✅ Design moderne avec animations fluides
- ✅ Navigation multi-pages avec React Router
- ✅ Formulaire de contact interactif
- ✅ Respect de la charte graphique KRABDATA
- ✅ Optimisé pour les performances

## 🎨 Charte Graphique

### Couleurs
- **Primaire (Navy Blue)**: `#1B2D5B` - Titres, header, footer, CTA
- **Secondaire (Beige)**: `#E8CFA5` - Fonds de sections, accents
- **Accent (Blue Light)**: `#3A5BA0` - Hover, liens interactifs
- **Fond principal**: `#FFFFFF`
- **Fond alternatif**: `#F5F5F5`
- **Texte**: `#333333`

### Typographies
- **Titres**: Montserrat (Bold/Black)
- **Corps de texte**: Open Sans (Regular)

## 📁 Structure du Site

### Pages principales

1. **Accueil** (`/`)
   - Hero section avec CTA
   - Notre Mission
   - Nos Domaines d'Expertise (5 services)
   - Chiffres clés avec compteurs animés
   - CTA final

2. **Nos Services** (`/services`)
   - Valorisation des Données (Data & IA)
   - Digitalisation & Automatisation (par direction métier)
   - Développement Web & Applicatifs
   - Cloud & Infrastructure

3. **Cas d'usage / Réalisations** (`/realisations`)
   - 5 cas d'usage détaillés
   - Contexte, Solution, Résultats, Technologies

4. **À propos** (`/a-propos`)
   - Histoire de l'entreprise
   - Vision et approche
   - Nos valeurs
   - Notre engagement

5. **Contact** (`/contact`)
   - Formulaire de contact
   - Coordonnées
   - Horaires d'ouverture

6. **Pages légales**
   - Mentions légales (`/mentions-legales`)
   - Politique de confidentialité (`/confidentialite`)

## 📱 Responsive Design

Le site est optimisé pour tous les écrans:
- **Mobile**: < 768px (menu hamburger, grille 1 colonne)
- **Tablette**: 768px - 1024px (grille 2 colonnes)
- **Desktop**: > 1024px (grille complète, navigation horizontale)
